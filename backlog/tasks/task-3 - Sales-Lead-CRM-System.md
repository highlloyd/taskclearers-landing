## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Build a sales lead CRM in the admin panel with pipeline management. Track leads through stages from initial contact to won/lost. Include Kanban board view for visual pipeline management. Integrate with existing email and notes systems.
<!-- SECTION:DESCRIPTION:END -->

## Acceptance Criteria
<!-- AC:BEGIN -->
- [ ] #1 Sales leads can be created, viewed, edited, and deleted
- [ ] #2 Lead data includes: company name, contact name, email, phone, estimated value, source, assigned salesperson
- [ ] #3 Pipeline stages: new, contacted, qualified, proposal, negotiation, won, lost
- [ ] #4 Kanban board view with drag-and-drop stage changes
- [ ] #5 Table/list view with filtering by stage, source, and assignee
- [ ] #6 Notes can be added to lead records
- [ ] #7 Activity log tracks all interactions and stage changes
- [ ] #8 Emails can be sent to leads using existing email system
- [ ] #9 Won/lost tracking with dates and reasons
<!-- AC:END -->
